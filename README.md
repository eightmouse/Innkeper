# Innkeeper

## Disclaimer
This app does NOT replace Armory in any way, shape or form, be respectful.

It a personal project to push my skills further while I play the game!
I like the Armory, so I though of a similar but more digestible idea to display basic infos and have it on your desktop basically.

## Description 
It's a minimal and simple World of Warcraft tracker to check basic informations such as:
- Dailies
- Weeklies
- World Quests
- And more to come!
